# MACTree mechanism profile

This directory is generated by `flood_local/build_mactree_profile.py`.

## Purpose

It extracts structural differences between the base FLOOD tree and the standalone `mactree/` material.
It does not enable mactree timing in the simulator.

## Strict paper policy

- This profile is evidence inventory only.
- No latency or speedup row may use MACTree effects until timing and value checks pass.
- Required evidence: RTL/testbench cycles, output values, and activity counters if power is claimed.
