# FLOOD RTL Boundary Calibration v4

## 定位

这是一个候选校准版本，用新补的边界 RTL 点修正 v3 公式。它提高了模型内部一致性，但因为边界点参与了拟合，所以还需要新的独立留出样本验证后才能作为论文性能模型使用。

## v4 修正

- 对 `k>1` 与 `group_size>4` 增加 `2.875*(k-1)*max(group_size-4,0)*(cout-1)` 交互项。
- 对 `k=1/group_size=16` 使用 `final_run = 56 + 19*max(cout-2,0)`。
- 多 Cin 仍暂按 `nonfinal = final_run - 3`，但 `group_size=16` 多 Cin RTL 已发现控制异常，不能用于论文性能数据。
