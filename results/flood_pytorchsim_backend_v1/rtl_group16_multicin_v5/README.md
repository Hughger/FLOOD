# FLOOD RTL Group16 Multi-Cin Calibration v5

## 目的

本报告使用修复后的 testbench，对 `group_size=16` 的多 Cin 情况进行 RTL 采样，用于建立 v5 高 group 多 Cin 校准项。

## 数据范围

- `k=1`
- `group_size=16`, `group_num=1`
- `cout=2/4/6/8/12/16`
- `cin_idx_total=2/4`
- `res_cols=res_rows=1`

所有样本 `x_count=0`，没有 0 周期 run。

## v5 规则

对 `k=1/group_size=16/res=1`：

```text
first_run = 19*cout + 15
middle_run = 53
final_run = 56
total = first_run + max(cin_idx_total-2, 0)*middle_run + final_run
```

- v5 平均绝对误差：0.0%，最大：0.0%

## 论文使用建议

这批数据可以支撑 `group_size=16` 多 Cin 的 RTL-aware v5 校准。但 `res_cols/res_rows>1` 的空间重复路径仍有 X 问题，不能混入同一个可信度等级。
